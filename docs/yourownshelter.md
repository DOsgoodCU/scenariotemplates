# Build your own Shelter Scenario

The Templates are designed to be tailored by the people who might use them.

This is the form used to tailor the Shelter Scenario.  

Please feel edit any or all of the text in the boxes to create the code for your own simulation. 

The email is used to identify the code specific to you as a user. If you dont want to share your email, 
you can enter something else but we wont be able to follow up if you want your simulation activated.

Note that all of the messages that are variables determined by the participants inputs are of course not editable by you, and are in the format %%phoneresponse%%.
